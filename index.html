HTML CHEAT SHEET

Hello world, in my attempt to have somewhere to reference in future when building web apps I decided to create an HTML Cheat Sheet

Table Tags

Tag Name Description
⁢table> Table The wrapper element for all HTML table
⁢thead> Table Head The set of rows defining the column headers in a table.
⁢tbody> Table Body The set of rows containing actual table data.
⁢tr> Table row The table row container.
⁢tdata> Table Data The table row container.
⁢tfoot> Table Foot The set of rows defining the footer in a table.

Table Attribute

Tag Name Description
colspan Column span Defines how many columns a td element should span.
rowspan Row span Defines how many rows a td element should span.

Other HTML Tags

Tag Name Description
⁢p> paragraph This tag defines a paragraph
⁢h1> Heading Defines a heading. the higher the number the smaller the heading size
